# boxdash-ui / git-keep

This project aims to build basic note dashboard functionality similar to GoogleKeep.

## Features
-close to save
-other note hotkeys(enter to add new, up /down arrow keys)
--background coloring
--searching / filtering / tagging

### Skipped features

no pics
